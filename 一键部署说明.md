# 🚀 训练日记 - 一键部署指南（无需 GitHub）

## 方案一：使用 Render 免费部署（推荐）

### 步骤 1：注册 Render 账号
1. 打开 https://render.com/
2. 点击 "Get Started for Free"
3. 选择用 Google 账号或邮箱注册（建议用 Gmail 或 Outlook 邮箱）

### 步骤 2：创建 Web Service
1. 登录后点击 "New +" 按钮
2. 选择 "Web Service"
3. 选择 "Upload your code"（上传代码）

### 步骤 3：上传代码并配置
1. 上传本文件夹压缩包（training-journal.zip）
2. 填写配置：
   - **Name**: training-journal
   - **Runtime**: Node
   - **Build Command**: `pnpm install && pnpm run build`
   - **Start Command**: `node server.js`
   - **Plan**: Free

3. 添加环境变量：
   - 点击 "Environment" 标签
   - 添加 `JWT_SECRET`，值填写一个随机字符串（比如 `my-secret-key-123456`）

4. 点击 "Create Web Service"

### 步骤 4：等待部署完成
- 部署大约需要 2-3 分钟
- 完成后会显示一个网址，比如 `https://training-journal-xxx.onrender.com`
- 这个网址就是你的训练日记网站！

## 方案二：本地部署（自己电脑运行）

### Windows 用户
1. 安装 Node.js：https://nodejs.org/ （下载 LTS 版本）
2. 解压本项目文件夹
3. 双击运行 `start.bat`
4. 打开浏览器访问 http://localhost:3000

### Mac/Linux 用户
1. 安装 Node.js 18+：
   ```bash
   # macOS
   brew install node
   
   # Ubuntu/Debian
   sudo apt-get install nodejs npm
   ```

2. 运行项目：
   ```bash
   cd training-journal
   chmod +x start.sh
   ./start.sh
   ```

3. 打开浏览器访问 http://localhost:3000

## 注意事项

1. **Render 免费版**：网站会在 15 分钟无访问后休眠，首次访问可能需要等待 30 秒唤醒
2. **数据存储**：所有数据保存在 Render 服务器的 SQLite 数据库中
3. **备份**：定期导出数据（可在网站设置中找到导出功能）

## 需要帮助？

如果遇到问题：
1. 检查 Node.js 版本是否 18+
2. 检查端口 3000 是否被占用
3. 查看 Render 的部署日志排查错误
